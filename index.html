Customer Churn Prediction

Overview: This project builds a machine learning pipeline to predict telecom customer churn. By analyzing historical usage patterns, the model identifies customers at risk of leaving, enabling proactive retention strategies.

Project Pipeline

  • Data Exploration (EDA): Analyzing patterns in customer demographics and usage.
  • Data Cleaning: Handling missing values and feature engineering.
  • Model Training: Implementing Random Forest and Logistic Regression models.
